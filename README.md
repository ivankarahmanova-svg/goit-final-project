# goit-final-project#
Personal Assistant

CLI-застосунок для роботи з адресною книгою та нотатками.

## Функціонал

### Контакти
- додавання контакту
- зміна телефону
- перегляд телефонів
- видалення контакту
- додавання дня народження
- перегляд дня народження
- список найближчих днів народження

### Нотатки
- додавання нотатки
- редагування нотатки
- видалення нотатки
- перегляд усіх нотаток
- пошук нотатки за назвою
- додавання тегів
- пошук за тегом
